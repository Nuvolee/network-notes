# ⚙️ 配置说明

- 1️⃣ 删除RIP 
- R2删除右侧RIP
```
  system-view
  rip 1
  undo network 23.0.0.0
```



- R3删除RIP，并宣告OSPF
```
  system-view
  undo rip 1

  ospf 1
  area 0
  network 23.1.1.0 0.0.0.255
  network 192.168.2.0 0.0.0.255
```


- 2️⃣ 路由引入

R2进行路由引入
```
  rip 1
  import-route ospf 1
```
