## 1️⃣ OSPF+RIP+互相引入

###  打通AR1 → R2 → AR3
  
- AR1
  ```
  system-view
  undo info enable
  sysname AR1
  
  int g0/0/0
  ip add 192.168.1.254 24
  int g0/0/1
  ip add 10.1.1.1 24
  ```
  
- R2
  ```
  system-view
  undo info enable
  sysname R2
  
  int g0/0/0
  ip add 10.1.1.2 24
  int g0/0/3
  ip add 11.1.1.2 24
  ```

- AR3
```
  system-view
  undo info enable
  sysname AR3
  int g0/0/0
  ip add 11.1.1.3 24
```

- 🔎测试：
R2-ping 10.1.1.1
R2-ping 11.1.1.3

### 配置OSPF和RIP

-AR1
```
  ospf 1
  area 0
  network 192.168.1.0 0.0.0.255
  network 10.1.1.0 0.0.0.255
```

- R2
```
  ospf 1
  area 0
  network 10.1.1.0 0.0.0.255
  network 11.1.1.0 0.0.0.255

  q
  q
  rip 1
  version 2
  network 11.0.0.0
```

- AR3
```
  rip 1
  version 2
  network 11.0.0.0
```

### OSPF和RIP互相引入
- R2
```
  ospf 1
  import-route rip 1
  rip 1
  import-route ospf 1
```

- 🔎 测试：

AR3-ping10.1.1.1
