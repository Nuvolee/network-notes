# 🖥️ 拓扑设计
<img width="1989" height="1281" alt="2026429end" src="https://github.com/user-attachments/assets/83a8a685-b93b-4a4e-bf36-692d03e82d7c" />

-  要求：
  
1. AR1、R2使用OSPF路由协议通信，R2、AR3之间使用RIP路由协议通信
2. SW1、SW2之间使用链路聚合使通信设备冗余，SW1作为网关，为下面的PC1、PC2分配IP地址
3. SW3、SW4运行VRRP协议使下面PC3、PC4网关冗余，
SW3、SW4与SW7之间使用链路聚合协议使链路冗余，虚拟IP:192.168.3.254
4. SW5、SW6分别作为下联设备的网关设备，提供出口服务
5. 将OSPF路由协议与RIP路由协议互相引入，使全网通信




# ⚙️ 配置实现

- 1️⃣PC网络配置

PC名|IP地址|子网掩码|网段
---|---|---|---
PC1|192.168.1.10|255.255.255.0|192.168.1.254|
PC2|192.168.2.10|255.255.255.0|192.168.2.254|
PC3|192.168.3.10|255.255.255.0|192.168.3.254|
PC4|192.168.3.20|255.255.255.0|192.168.3.254|
PC5|192.168.5.10|255.255.255.0|192.168.5.254|
PC6|192.168.6.10|255.255.255.0|192.168.6.254|


- 2️⃣ 打通AR1 → R2 → AR3
- AR1
  system-view
  undo info enable
  sysname AR1
  
  int g0/0/0
  ip add 192.168.1.254 24
  int g0/0/1
  ip add 10.1.1.1 24
  
- R2
  system-view
  undo info enable
  sysname R2
  
  int g0/0/0
  ip add 10.1.1.2 24
  int g0/0/3
  ip add 11.1.1.2 24
- 

- AR3
  system-view
  undo info enable
  sysname AR3
  int g0/0/0
  ip add 11.1.1.3 24
-
