# 🖥️ 拓扑设计
<img width="1914" height="1248" alt="2026429end" src="https://github.com/user-attachments/assets/f117d19d-b20f-4662-9794-1461f02c71b5" />



-  要求：
  
1. AR1、R2使用OSPF路由协议通信，R2、AR3之间使用RIP路由协议通信
2. SW1、SW2之间使用链路聚合使通信设备冗余，SW1作为网关，为下面的PC1、PC2分配IP地址
3. SW3、SW4运行VRRP协议使下面PC3、PC4网关冗余，
SW3、SW4与SW7之间使用链路聚合协议使链路冗余，虚拟IP:192.168.3.254
4. SW5、SW6分别作为下联设备的网关设备，提供出口服务
5. 将OSPF路由协议与RIP路由协议互相引入，使全网通信

# 路由器配置
> 主要配置：接口IP，路由协议（OSPF/RIP）


## ⚙️ AR1配置

- 接口IP
```
  system-view
  undo info enable
  sysname AR1

  int g0/0/0
  ip add 10.1.1.1 24
  int g0/0/1
  ip add 11.1.1.1 24
  q
```


- OSPF

```
  ospf 1
  area 0
  network 10.1.1.0 0.0.0.255
  network 11.1.1.0 0.0.0.255
```




## ⚙️ R2配置
- 接口IP
```
  system-view
  undo info enable
  sysname R2

  int g0/0/0
  ip add 11.1.1.2 24
  int g0/0/1
  ip add 13.1.1.1 24
  int g0/0/2
  ip add 14.1.1.1 24
  int g0/0/3
  ip add 12.1.1.1 24
```

- 配OSPF：注意，R2的g0/0/3是右连AR3的，不要宣
```
  ospf 1
  area 0
  network 11.1.1.0 0.0.0.255
  network 13.1.1.0 0.0.0.255
  network 14.1.1.0 0.0.0.255
  q
```

- 配RIP
```
  rip 1
  version 2
  network 12.0.0.0
```

- 互相引入
```
  ospf 1
  import-route rip 1
  q
  rip 1
  import-route ospf 1
```

# ⚙️ 交换机配置
> 主要配置：
- VLAN
- VLANIF
- 接口配置
- Eth-Trunk链路聚合
- DHCP
- 协议


## ⚙️ SW1配置
- VLAN+VLANIF
```
  system-view
  undo info enable
  sysname SW1

  vlan batch 10 20 100

  int Vlanif 10
  ip add 192.168.1.254 24
  q
  int Vlanif 20
  ip add 192.168.2.254 24
  q
  int Vlanif 100
  ip add 10.1.1.2 24
```


- 上连AR1-access
```
  int Vlanif 100
  ip add 10.1.1.2 24
```



- 下连SW2-链路聚合
```
  int Eth-Trunk 1
  mode lacp-static
  int g0/0/2
  eth-trunk 1
  int g0/0/3
  eth-trunk 1
  q

  int Eth-Trunk 1
  port link-type trunk
  port trunk allow-pass vlan 10 20
  q
```


- 启动DHCP，SW1作为网关，给PC1、2分配地址
```
  dhcp enable

  ip pool vlan10
  network 192.168.1.0 mask 255.255.255.0
  gateway-list 192.168.1.254
  dns-list 8.8.8.8
  q

  ip pool vlan20
  network 162.168.2.0 mask 255.255.255.0
  gateway-list 192.168.2.254
  dns-list 8.8.8.8
  q
```


- 开启分配
```
  int Vlanif 10
  dhcp select global
  int Vlanif 20
  dhcp select global
  q
```


- 开启OSPF
```
  ospf 1
  area 0
  network 10.1.1.0 0.0.0.255
  network 192.168.1.0 0.0.0.255
  network 192.168.2.0 0.0.0.255
```
