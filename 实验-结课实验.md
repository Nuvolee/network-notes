-  要求：
  
1. AR1、R2使用OSPF路由协议通信，R2、AR3之间使用RIP路由协议通信
2. SW1、SW2之间使用链路聚合使通信设备冗余，SW1作为网关，为下面的PC1、PC2分配IP地址
3. SW3、SW4运行VRRP协议使下面PC3、PC4网关冗余，
SW3、SW4与SW7之间使用链路聚合协议使链路冗余，虚拟IP:192.168.3.254
4. SW5、SW6分别作为下联设备的网关设备，提供出口服务
5. 将OSPF路由协议与RIP路由协议互相引入，使全网通信


 VLAN →  Eth-Trunk →  Trunk →  VRRP

## 1️⃣ 路由配置

###  打通AR1 → R2 → AR3
  
- AR1
  ```
  system-view
  undo info enable
  sysname AR1
  
  int g0/0/0
  ip add 192.168.1.254 24
  int g0/0/1
  ip add 10.1.1.1 24
  ```
  
- R2
  ```
  system-view
  undo info enable
  sysname R2
  
  int g0/0/0
  ip add 10.1.1.2 24
  int g0/0/3
  ip add 11.1.1.2 24
  ```

- AR3
```
  system-view
  undo info enable
  sysname AR3
  int g0/0/0
  ip add 11.1.1.3 24
```

- 🔎测试：
R2-ping 10.1.1.1
R2-ping 11.1.1.3

### 配置OSPF和RIP

-AR1
```
  ospf 1
  area 0
  network 192.168.1.0 0.0.0.255
  network 10.1.1.0 0.0.0.255
```

- R2
```
  ospf 1
  area 0
  network 10.1.1.0 0.0.0.255
  network 11.1.1.0 0.0.0.255

  q
  q
  rip 1
  version 2
  network 11.0.0.0
```

- AR3
```
  rip 1
  version 2
  network 11.0.0.0
```

### OSPF和RIP互相引入
- R2
```
  ospf 1
  import-route rip 1
  rip 1
  import-route ospf 1
```

- 🔎 测试：
AR1-ping 11.1.1.3
AR3-ping 10.1.1.1

## 2️⃣ VLAN配置
- SW1
```
  system-view
  undo info enable
  sysname SW1

  vlan batch 10 20

  int Eth-Trunk 1
  mode lacp-static
  port link-type trunk
  port trunk allow-pass vlan 10 20

  int g0/0/2
  eth-trunk 1
  int g0/0/3
  eth-trunk 1

```


- SW2
```
  system-view
  undo info enable
  sysname SW2

  vlan batch 10 20

  int g0/0/3
  port link-type access
  port default vlan 10
  int g0/0/4
  port link-type access
  port default vlan 20

  int Eth-Trunk 1
  mode lacp-static
  port link-type trunk
  port trunk allow-pass vlan 10 20

  int g0/0/1
  eth-trunk 1
  int g0/0/2
  eth-trunk 1

```



- VLANIF
- SW1
```
  int Vlanif 10
  ip add 192.168.1.254 24
  q
  int Vlanif 20
  ip add 192.168.2.254 24
```

- SW7 ↔ SW3（Eth-Trunk 1）


- 配VRRP并且接入E-trunk
- SW3
```
  int Vlanif 30
  ip add 192.168.3.1 24

  vrrp vrid 1 virtual-ip 192.168.3.254
  vrrp vrid 1 priority 120

  q
  int Eth-Trunk 1
  mode lacp-static
  port link-type trunk
  port trunk allow-pass vlan 30
  int g0/0/2
  eth-trunk 1
  int g0/0/3
  eth-trunk 1
```

- SW4
```
  system-view
  ip add 192.168.3.2 24

  int Vlanif 30
  vrrp vrid 1 virtual-ip 192.168.3.254
  vrrp vrid 1 priority 100

  q
  int Eth-Trunk 2
  mode lacp-static
  port link-type trunk
  port trunk allow-pass vlan 30
  int g0/0/2
  eth-trunk 2
  int g0/0/3
  eth-trunk 2
```
