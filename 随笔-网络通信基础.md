# 🌐 3. 网络基本组成

计算机网络主要由以下四部分构成：

| 组成 | 功能 | 示例 |
|------|------|------|
| 终端设备 | 数据发送与接收 | PC、服务器、手机 |
| 传输介质 | 数据传输通道 | 光纤、双绞线 |
| 网络设备 | 数据转发与控制 | 交换机、路由器 |
| 通信协议 | 数据通信规则 | TCP/IP、HTTP |

 协议可以理解为“通信语言”，就像人与人交流需要语言，所有设备必须遵守统一规则才能通信。
IP：互联网协议（如QQ，用的QICQ）




# 🌐 4. 网络的作用

网络的核心作用是：**实现数据的传输与资源共享**

典型应用场景：
- 即时通讯（微信、QQ）
- 视频播放（B站、抖音）
- 在线游戏
- 云服务与远程办公


  

# 📦 5. 数据封装与解封装

- 数据封装过程（发送方向）
数据从上到下依次封装：
应用层数据 → 传输层加TCP头 → 网络层加IP头 → 数据链路层加MAC头 → 物理层发送比特流 

- 数据解封装过程（接收方向）
物理层 → 数据链路层 → 网络层 → 传输层 → 应用层




# 🧱 6. OSI与TCP/IP模型对比

| OSI七层模型 （开放式系统互联参考模型）| TCP/IP 协议栈 |
|-------------|------------|
| 应用层 | 应用层 |
| 表示层 | ❌ |
| 会话层 | ❌ |
| 传输层 | 传输层 |
| 网络层 | 网络层 |
| 数据链路层 | 网络接口层 |
|物理层| 网络接口层 |

 TCP/IP是OSI的工程简化模型。
